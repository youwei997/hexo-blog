react 内置hook

useState,useEffect

watchEffect第二个参数

  • 第二个没有参数：组件初始调用一次，组件每次更新时调用（setState时）
  • 第二个参数是空数组：组件初始调用一次
  • 第二个参数是数组，组件初始调用一次，数组里值改变时重新调用

watchEffect清除副作用

清除副作用适用于清除定时器，子组件有定时器，在父组件卸载子组件时（条件判断），子组件如果没有在watchEffect清除定时器，定时器会一直存在

  2. 可以获取 dom 的属性和方法
