## 默认

```go
package main

import "fmt"

func main() {
	for i := 0; i < 10; i++ {
		fmt.Println(i)
	}
}

```

## 循环数组/切片

```go
package main

import "fmt"

func main() {
	var slice1 = []int{1, 2, 3}
	for i := 0; i < len(slice1); i++ {
		fmt.Println(slice1[i])
	}
}

```

## 死循环

```go
package main

import "fmt"

func main() {
	for {
		fmt.Println("死循环")
	}
}

```
